# ECON-2080-SPRING2026

**PhD Macroeconomics — Spring 2026**

This repository contains the code and results for the problem sets assigned in **ECON 2080**, a PhD-level Macroeconomics course. The course covers three main thematic areas:

- **Labor Markets** — search-and-matching models, wage determination, and unemployment dynamics
- **Distribution Economics** — heterogeneous-agent frameworks, income and wealth inequality, and distributional dynamics
- **Growth Models** — neoclassical and endogenous growth theory, capital accumulation, and long-run dynamics

## Topics & Problem Sets

### Problem Set 1 — Labor Markets
Covers models of the labor market, including:
- Search-and-matching models (Mortensen-Pissarides framework)
- Wage bargaining and Nash bargaining solutions
- Steady-state unemployment and vacancy rates
- Comparative statics with respect to separation rates, matching efficiency, and bargaining power

**Key results:** Equilibrium Beveridge curves, job-finding and filling rates as functions of market tightness, and policy experiments (e.g., unemployment insurance).

---

### Problem Set 2 — Distribution Economics
Covers heterogeneous-agent macroeconomic models, including:
- Incomplete-markets (Bewley-Huggett-Aiyagari) economies
- Stationary wealth and income distributions
- Gini coefficients and Lorenz curves
- The role of idiosyncratic income risk and borrowing constraints on inequality

**Key results:** Stationary asset and consumption distributions, Lorenz curves for wealth and income, and welfare comparisons across policy experiments.

---

### Problem Set 3 — Growth Models
Covers neoclassical and endogenous growth theory, including:
- Solow-Swan growth model and steady-state capital-output ratios
- Ramsey-Cass-Koopmans optimal growth model
- Endogenous growth (AK model and knowledge-spillover models)
- Transitional dynamics and convergence

**Key results:** Balanced growth paths, phase diagrams for capital and consumption, and numerical simulations of transitional dynamics under different parameter calibrations.

## Code

All code is written in **MATLAB** and/or **Python**. Each script is documented with inline comments that explain the economic model and numerical methods used. Key methods include:

- **Value Function Iteration (VFI)** — for solving dynamic programming problems
- **Euler Equation methods** — policy function iteration and endogenous grid methods
- **Finite-difference methods** — for solving Hamilton-Jacobi-Bellman equations
- **Simulation** — Monte Carlo and panel-data simulation of model economies

## Results

Figures and tables are saved in each problem set's `results/` folder. Typical outputs include:

- Equilibrium policy functions (consumption, savings, labor supply)
- Stationary distributions (wealth, income, employment)
- Comparative statics tables
- Phase diagrams and transitional dynamics plots

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/jorgeluis8ar/ECON-2080-SPRING2026.git
   cd ECON-2080-SPRING2026
   ```

2. Navigate to the problem set of interest:
   ```bash
   cd ProblemSet1/code
   ```

3. Run the main script (MATLAB example):
   ```matlab
   main.m
   ```
   Or for Python:
   ```bash
   python main.py
   ```

4. Results are saved automatically to `../results/`.
